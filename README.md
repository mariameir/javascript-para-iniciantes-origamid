# Portfólio de JavaScript para Iniciantes

Bem-vindo ao meu portfólio de JavaScript para iniciantes! Aqui você encontrará os scripts de JavaScript que fiz durante o curso da [ORIGAMID](https://www.origamid.com/)
.

## Pastas e Assuntos

1. **Variáveis:** Nesta pasta, você encontrará scripts que abordam o conceito de variáveis e como declará-las em JavaScript.

2. **Tipos de Dados:** Aqui estão os scripts relacionados aos diferentes tipos de dados em JavaScript, como strings, números, booleanos, arrays e objetos.

3. **Números e Operadores:** Nesta pasta, você encontrará scripts que exploram os operadores matemáticos e aritméticos em JavaScript.

4. **Boolean e Condicionais:** Aqui estão os scripts que lidam com valores booleanos e estruturas condicionais como `if`, `else` e `switch`.

5. **Funções:** Nesta pasta, você encontrará scripts que abordam o conceito de funções em JavaScript e como criar e invocá-las.

6. **Objetos:** Aqui estão os scripts relacionados a objetos em JavaScript, incluindo criação, manipulação e acesso a propriedades.

7. **Tudo é Objeto:** Nesta pasta, você encontrará scripts que explicam como praticamente tudo em JavaScript é um objeto.

8. **Arrays e Loops:** Aqui estão os scripts que abordam arrays em JavaScript e como percorrer seus elementos usando loops.

9. **Atribuição e Ternário:** Nesta pasta, você encontrará scripts que exploram operadores de atribuição e o operador ternário em JavaScript.

Fique à vontade para explorar cada pasta e seus respectivos scripts para aprender mais sobre JavaScript!

## Como Utilizar

1. Clone este repositório em sua máquina local usando o seguinte comando:

    `git clone https://github.com/mariameiro/javascript-para-iniciantes-origamid`


2. Navegue até a pasta do script que deseja executar.

3. Abra o arquivo `.html` no seu navegador para ver o resultado da execução do script.

4. Sinta-se à vontade para fazer alterações e experimentar os conceitos abordados nos scripts.
